How does JavaScript compare to HTML and CSS?

Javascript allows for complex logic, including control flows and loops, whereas HTML and CSS are more descriptive languages. Javascript can be used to implement interative features on a website.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'

Control flow process starts with assessing a situation, then depending on this situation, taking one or more actions.

An example of control flow in everyday life:

  • Assessing the situation: is the fridge empty?
  • Depending on the situation, take action
    • If the fridge is full: Decide what to eat and cook dinner
    • Else it means the fridge is empty: Decide where to eat out, get in your car and go eat at the restaurant

Another example of control flow:

  • Assessing the situation: Ask your friends if they want to watch a movie, play boardgames, or go out?
  • Depending on the situation, take action
    • In case they want to watch a movie: Decide what movie to watch and put the movie on
    • In case thet want to play boardgames: Decide what boardgame to play, get the boardgame out and start playing
    • In case they want to go out: Decide where to go and how to dress, they go out
  • After your activity, resume to the next step is your daily process: go to sleep - your allowed to take a break

A Loop is the process of repeating the same action until reaching a defined situation.

An example of loop in every day life:

  • Action to be repeated: manually clean the wine glasses
  • Until reaching a defined situation: all glasses are clear
  • In this case, the number of glasses is predefined, hence we know how many times we will need to repeat the action of cleaning a glass

Another example of loop in every day life:

  • Action to be repeated: scrub the stain of red wine glasses you've done on the carpet
  • Until reaching a defined situation: Stain has disappeared
  • In this case, you don't know exactly how much scrubbing is required in advance

Explain the difference between accessing data from arrays and object literals

Arrays and Objects are both used to store data or values of different types (string, numbers, booleans, arrays, objects):

  • An array is a group of ordered values, where each value is represented by an index: a number starting at 0, and increasing by one for the next value in the array
  • An objet is a group of key - value pairs, where a key can be any number or word
Accessing information contained in an array is done by referring to the position (index) of the piece of data in the array. Accessing information contained in an object is done by referring to the key representing the data.

Explain what functions are and why they are useful

A function is defined as a set of actions. These actions are regrouped in a high level concept, the function, and can be reused throughout your code. It limits code repetition: remember D.R.Y. A function is defined as taking a set of parameters which value will be taken into account during the execution of the actions. It can also return an output from its execusion.

Here is an everyday life example: "Making a smoothie" is your function.

Making a smoothie requires the execution of several actions: prepare the fruits, get the right quantity of milk and other ingredients ready, pour all the ingredients in the mixer, and mix. Everytime you make a smoothie, you will take the same actions, although you have some variety in the type of smoothie you can make!

You can make a smoothie with different ingredients, for instance: Banana, almond milk and cashew nuts. Ingredients are your function parameters, banana, almond milk and cashew nuts are the arguments used to execute the function.

...and the output is a delicious smoothie!
