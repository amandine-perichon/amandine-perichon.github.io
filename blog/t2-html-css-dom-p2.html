What is grid based design?

Grid based design is the practice of dividing your webpage into a grid, using different arrangements of number of columns and widths. For example:

  • 1 row of 100%
  • 3 columns of 33% width each
  • 2 columns of 25% width and 1 column of 50%

To each type of column corresponds a CSS class that can be applied to different HTML elements, and reused throughout the website.

Many websites use a grid with 12 columns as this gives quite a lot of flexibility. More than 12 columns however would make for too narrow columns.

What's all the hype about responsive webpages?

Responsive design means developing websites which will resize and adapt to different sizes of screen (different resolutions) and types of devices, ensuring the website is still legible and is looking great.

With the multitude of devices out there - desktops, laptops, phones, tables of different sizes - no wonder there is a hype about responsive webpages!

The chrome dev tools have a cool feature that lets you view a webpage in a different screen size.

What is semantic structure?

When using semantic structure to build a webpage, one uses elements that have a defined meaning. For instance, the webpage could contain a header, section, and footer. Users will expect the top of a website to contain a header with a website name and a navbar.

HTML5 includes tags called semantic elements, which give information about the type of content these elements of the website will contain: header, footer, article, aside.

Previous versions of HTML achieved semantic structure through assigning a class to elements which are not specific to a type of information. For example <div class="header"> would containt a header, where div elements are not specific to a type of content.
